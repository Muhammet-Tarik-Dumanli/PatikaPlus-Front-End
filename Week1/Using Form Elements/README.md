# Basit HTML Formu Uygulaması

Bu proje, HTML ve temel CSS kullanılarak hazırlanmış basit bir form uygulamasıdır. Form içerisinde kullanıcıdan ad, soyad, e-posta, parola, cinsiyet, favori meyve ve mesaj bilgileri alınmaktadır. CSS ile form sade bir şekilde stillendirilmiştir.

## 💡 Amaç

Bu uygulamanın amacı, HTML form elemanlarını tanımak ve basit CSS kuralları ile nasıl stillendirilebileceğini öğrenmektir.

## 🛠️ Kullanılan Teknolojiler

- **HTML5** – Form yapısını oluşturmak için
- **CSS3** – Formu basit şekilde stilize etmek için

## 📋 Form Elemanları

Formda yer alan temel bileşenler:

- `input[type="text"]` — Ad ve Soyad girişi
- `input[type="email"]` — E-posta girişi
- `input[type="password"]` — Parola girişi (minimum 8 karakter)
- `input[type="radio"]` — Cinsiyet seçimi (Erkek / Kadın)
- `select` — Favori meyve seçimi
- `textarea` — Forum hakkındaki düşüncelerin girileceği alan
- `input[type="submit"]` — Formu göndermek için düğme

## 🎨 CSS Özellikleri

- Form kutusu sayfanın ortasında yer alır.
- Tüm giriş alanları %100 genişliktedir ve okunabilirlik için iç boşluk (padding) içerir.
- `submit` butonu yeşil renkte olup, üzerine gelindiğinde (hover) tonu biraz koyulaşır.
- Etiketler (`label`) ve giriş alanları arasında boşluklar vardır.


## 🚀 Nasıl Kullanılır?

1. Dosyayı bir klasöre indirin.
2. `index.html` dosyasını bir tarayıcıda açın.
3. Formu doldurarak "Gönder" butonuna tıklayın.

> **Not:** Formun `action` kısmı "README.md" olarak ayarlanmıştır. Gerçek bir sunucuya veri gönderimi yapılmaz, sadece örnek amaçlıdır.
